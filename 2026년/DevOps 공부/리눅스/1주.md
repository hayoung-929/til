# Week 1 - Day 1: SSH 원격 서버 실습 환경 구성

## 1. SSH 기본 개념

- **SSH (Secure Shell)**: 암호화된 채널로 원격 서버에 접속하는 프로토콜 (포트 22)
- 학교 서버에 접속하면 → 서버의 리눅스 쉘을 **내 터미널에서 직접 조작**하는 것

```
내 PC 터미널  ──── SSH (암호화) ────▶  학교 Ubuntu 서버
  (클라이언트)                          (원격 호스트)
```

---

## 2. SSH 접속 확인 및 기본 정보 파악

접속 후 아래 명령어로 서버 환경부터 파악한다.

```bash
# 현재 로그인한 사용자 확인
whoami

# 서버 OS 버전 확인
cat /etc/os-release

# 현재 내 위치 (홈 디렉토리 확인)
pwd

# 서버 호스트명 확인
hostname

# 현재 접속 중인 사용자 목록
who

# 서버 uptime (부팅 후 경과 시간)
uptime
```

---

## 3. 홈 디렉토리 실습 환경 구성

**주의**: 학교 서버는 공유 환경이므로 반드시 **본인 홈 디렉토리 안에서만** 작업한다.

```bash
# 홈 디렉토리 이동
cd ~

# 실습용 디렉토리 구조 생성
mkdir -p ~/linux-practice/week1/day1

# 생성 확인
ls -la ~/linux-practice/

# 실습 디렉토리로 이동
cd ~/linux-practice/week1/day1
```

---

## 4. 파일시스템 탐색 실습

```bash
# 루트 디렉토리 구조 확인
ls /

# 주요 디렉토리 확인
ls /etc       # 설정 파일들
ls /var/log   # 로그 파일들
ls /home      # 사용자 홈 디렉토리 목록 (다른 학생 계정도 보일 수 있음)

# 디스크 사용량 확인
df -h

# 현재 디렉토리 용량 확인
du -sh ~
```

---

## 5. 파일 생성 및 조작 실습

```bash
cd ~/linux-practice/week1/day1

# 파일 생성
touch memo.txt log1.txt log2.txt

# 내용 작성 (nano 에디터)
nano memo.txt
# → 내용 입력 후 Ctrl+O (저장), Enter, Ctrl+X (종료)

# 파일 내용 확인
cat memo.txt

# 파일 복사
cp memo.txt memo_backup.txt

# 파일 이름 변경
mv memo_backup.txt memo_bak.txt

# 현재 디렉토리 파일 목록 (상세)
ls -lh
```

---

## 6. 권한 확인 실습

```bash
# 파일 권한 확인
ls -la

# 실행 권한 부여
touch hello.sh
echo 'echo "Hello, Linux!"' > hello.sh
chmod +x hello.sh

# 실행
./hello.sh

# 권한 숫자로 변경
chmod 644 memo.txt    # rw-r--r--
chmod 755 hello.sh    # rwxr-xr-x
```

## 7. SSH 세션 관리 팁

| 상황 | 명령어 |
|------|--------|
| 접속 종료 | `exit` 또는 `Ctrl+D` |
| 실행 중 명령어 강제 중단 | `Ctrl+C` |
| 화면 지우기 | `clear` 또는 `Ctrl+L` |
| 이전 명령어 불러오기 | `↑` 방향키 |
| 명령어 자동완성 | `Tab` 키 |
| 접속이 끊겼을 때 | 터미널에서 ssh 명령어 재실행 |
